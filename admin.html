GitHub トークンの取得方法
ブラウザで下記 URL を直接開く：
https://github.com/settings/tokens

「Generate new token (classic)」をクリック
→ Scope の repo にチェック
→ 「Generate token」→ 表示されたトークンをコピー

※ トークンはブラウザに保存されるため、次回以降は入力不要
