写真の差し替え方法

  1. 差し替えたい写真の「テスト表示」ボタンを押し、新しい画像ファイルを選択すると並べて確認できます。
  2. 「パスをコピー」で保存先パスをコピーしてください。
  3. 新しい画像ファイルを同じファイル名にリネームし、コピーしたパスと同じ場所に配置します。
  4. GitHub リポジトリにプッシュすれば自動的にサイトへ反映されます（GitHub Pages の場合、数分かかります）。
  5. ファイル名を変える場合は、対応する HTML ファイル内の src 属性も合わせて変更してください。
